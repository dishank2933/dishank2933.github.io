# Project Images

Add your project screenshots here with multiple images per project for the slideshow feature.

## Image Naming Convention

For each project, add up to 3 images with these naming patterns:

### VB24 Platform
- `vb24.png` - Main screenshot
- `vb24-2.png` - Second screenshot
- `vb24-3.png` - Third screenshot

### BlueMoon Invoice Parser
- `bluemoon.png` - Main screenshot
- `bluemoon-2.png` - Second screenshot
- `bluemoon-3.png` - Third screenshot

### Ziro Automation
- `ziro.png` - Main screenshot
- `ziro-2.png` - Second screenshot
- `ziro-3.png` - Third screenshot

### Challan Office
- `challan-office.png` - Main screenshot
- `challan-office-2.png` - Second screenshot
- `challan-office-3.png` - Third screenshot

## Image Guidelines

- **Format**: PNG or JPG
- **Recommended size**: 1200x600px (16:9 aspect ratio)
- **File size**: Keep under 500KB for faster loading
- **Content**: Show different features or views of your project
- **Slideshow**: Images will automatically rotate every 4 seconds
- **Navigation**: Users can click arrows or dots to navigate manually

## Features

✨ **Auto-play**: Slides change automatically every 4 seconds
✨ **Manual navigation**: Arrow buttons and dot indicators
✨ **Pause on hover**: Slideshow pauses when user hovers over images
✨ **Fallback**: If only one image exists, slideshow won't initialize

## Adding Images

1. Save your screenshots in this folder
2. Rename them to match the expected filenames above
3. Commit and push to GitHub:
   ```bash
   git add images/projects/
   git commit -m "Add project screenshots"
   git push
   ```

If an image is missing, the website will show a placeholder with a code icon.
